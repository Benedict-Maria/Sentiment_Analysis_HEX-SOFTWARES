The dataset used in this project contains tweets or social media text along with associated user and regional information. It is designed for sentiment analysis, aiming to classify text as positive, negative, or neutral. The dataset includes the following columns:

textID – A unique identifier for each text entry, ensuring every record can be referenced independently.

text – The main content of the tweet or user-generated message, which is the primary input for sentiment analysis.

sentiment – The target label for classification, indicating whether the text conveys a positive, negative, or neutral sentiment.

Time of Tweet – The timestamp indicating when the text was posted, which can be used for time-based trend analysis.

Age of User – The age of the user who posted the text, allowing exploration of sentiment patterns across different age groups.

Country – The country of origin of the user, useful for geographical sentiment analysis.

Population-2020 – The population of the user’s country in 2020, providing context for regional sentiment distribution.

Land Area – Total land area of the country, useful in calculating population density or demographic insights.

Density – The population density of the country (people per square kilometer), which can be analyzed in relation to sentiment patterns.
